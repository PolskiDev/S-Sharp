S# PROGRAMMING LANGUAGE EXTENSION FOR VSCODE
============================================

1. Unrar (S_Sharp-VSCode-Extension.rar)

2. Copy (ss-lang) to (VSCode Extensions
   Folder) of your Operating System.

======================================================================================

(VSCode Extensions Folder):

======================================================================================
(Windows): "C:\Program Files\Microsoft VS Code\resources\app\extensions\"
*DIR: "Microsoft VS Code\resources\app\extensions\"


(Linux): $HOME/.vscode/extensions/
$HOME: /home/<username>


(macOS): "/Applications/Visual Studio Code.app/Contents/Resources/app/extensions/"

======================================================================================

Note: You can also build Visual Studio Code from sources for your operating system
or a custom hardware, such as: Code OSS for Raspberry Pi or OrangePi.
